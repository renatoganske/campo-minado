# Campo Minado

Projeto desenvolvido durante o curso de Java da Cod3r, ministrado pelo professor Leonardo Leitão. Trata-se do famoso jogo **Campo Minado**.
Nesse projeto pudemos revisitar e consolidar diversos conceitos, como: 
	* Criação de Classes
	* Encapsulamento
	* Recursividade
	* Orientação a Objeto
	* Lambda Expression
	* Streams
	* Tratamento de Erros
	* Testes 
	
O jogo deve ser executado no próprio console.

Recomendações:
	* Na classe Aplicação.java, onde o tabuleiro é instanciado, você pode definir o tamanho dele e a quantidade de bombas do jogo.
	Essa definição acontece na seguinte sequência: quantidade de linhas, quantidade de colunas, quantidade de bombas.
	
	* Para jogar, você deve informar qual campo quer abrir ou marcar, para isso deve informar o número da linha seguido do número da coluna;
	* Posteriormente, você deve informar se quer abrir ou (Des)Marcar o campo.
	Agora é só se divertir!
